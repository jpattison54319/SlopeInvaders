# Slope Invaders Agent Design Docs

These files are intended to guide AI coding/design agents working on **Slope Invaders**, a browser-based educational game for learning linear equations.

## Files

- `00-agent-principles.md` — evidence-grounded rules every agent should follow.
- `01-learning-design.md` — learning objectives, campaign scaffolding, feedback, and reflection.
- `02-adaptivity-personalization.md` — adaptive supports, teacher customization, and student agency.
- `03-gamification-multiplayer.md` — XP, badges, stars, multiplayer, and competition guardrails.
- `04-ui-audio-visual-design.md` — interface, cognitive load, aesthetics, story, and accessibility.
- `05-prototype-scope-zone-1.md` — concrete prototype scope for building Zone 1.
- `06-zone-2-intercepts.md` through `10-zones-6-8-plan.md` — implemented zone
  pedagogy and puzzle constraints.
- `10-classroom-cloud.md` — classroom sync and live Versus architecture.
- `11-arcade-mode.md` — endless Arcade timing, collision, scoring, records, and
  learning-design guardrails.
- `sources.md` — source notes and bibliography-style references from the readings.

## One-sentence design rule

Every feature must support a learning behavior: understanding slope, y-intercept, negative slope, quadrants, line-through-two-points reasoning, or strategic graph interpretation.
